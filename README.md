# Electric-Vehicle-Data-Analysis

## 📊 Overview
This project presents an interactive Electric Vehicle (EV) data analysis dashboard created using Tableau. The dashboard provides insights into EV growth, vehicle distribution, electric range, and manufacturer performance across different states.

## 🛠 Tools Used
- Tableau
- Excel
- Data Visualization
- Data Analytics

## 🎯 Objectives
- Analyze the growth of electric vehicles over the years
- Compare BEV (Battery Electric Vehicles) and PHEV (Plug-in Hybrid Electric Vehicles)
- Identify top EV manufacturers and models
- Study state-wise EV distribution
- Understand CAFV eligibility trends


## 📈 Key Metrics
- **Total Vehicles:** 1,50,456  
- **Average Electric Range:** 67.88 miles  
- **Total BEV Vehicles:** 1,16,785 (77.62%)  
- **Total PHEV Vehicles:** 33,671 (22.38%)


## 📊 Dashboard Insights

### 🚘 EV Growth Trend
- EV registrations have increased significantly over time
- Peak growth observed around 2023
- Slight drop in the latest year (2024)

### 🗺 State-wise Distribution
- EV adoption varies across different states
- Some states show significantly higher EV usage

### 🏭 Top Manufacturers
- Tesla leads with the highest number of vehicles
- Nissan and Chevrolet are also major contributors

### 🚙 Top Models
- Tesla Model Y and Model 3 are the most popular
- Other models like Nissan Leaf and Chevrolet Bolt EV also contribute

### 🔋 CAFV Eligibility
- Majority vehicles fall under CAFV eligible and unknown categories
- Smaller percentage of vehicles are not eligible

## 🎛 Filters Available
- CAFV Eligibility  
- EV Type  
- Model  
- State
